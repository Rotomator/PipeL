Face Rigger

Workflow:
  • Press Import Face Rig Template
  • Move Locators or curve to the place you want to be on the face
  • If you need to add extra Curves or Locators
If Extra Curve is added:
  • Select Curve
  • Click Add Attributes to Curve
  • rename Curve to finish with _crv
  • parent curve to base_head_rig_grp
If Extra Locators is added:
  • rename Locator to finish with _pnt
  • parent to base_head_rig_grp
  • Select Mesh to Deform (Generally the main head)
  • Click 2.<<< Button
If there are vertices you don't want to deform, like Eyelids:
  • Select vertices
  • Click <<< Button
  • Click 3.Create SoftMods
  • Edit Soft weights
  • Select Mesh To Skin( Generally a head copy to use later as Blendshape for the main head)
  • Click 4.<<<
If Mesh To Skin has a skinCluster:
  • Click <<< to use this skinCluster
  • Click Create Face Rig
